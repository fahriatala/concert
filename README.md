## Requirements

This version of application requires the following dependencies

| Name          | Details                  |
|---------------|--------------------------|
| IDE           | Intellij IDE             |
| Java Version  | Java 8                   |
| Framework     | Spring Boot              |
| Database      | MySQL                    |
